Les portraits

La saisie
=========

La création d'un nouveau portrait se fait en choisissant ``Portrait`` dans le menu ``Ajout d'un élément`` :

.. image:: portrait1.png

Plusieurs onglets sont présentés :

.. image:: portrait2.png

.. warning:: Un champ qui porte un point rouge est un champ obligatoire. Il n'est pas possible
   de valider le formulaire sans avoir saisi une valeur dans ce champ. D'autre part, certains champs
   ne sont pas obligatoires mais le contenu est vérifié. Par exemple, pour les URL des pages
   WEB : elles ne sont pas obligatoires, mais si l'URL n'est pas valide, le formulaire ne peut
   être validé.
